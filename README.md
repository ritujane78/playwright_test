# Playwright Testing Framework (Java)

This project demonstrates various testing techniques using **Playwright** with **JUnit**, **Cucumber (BDD)**, and **Allure reporting (with screenshots)**.  
It is designed to provide a complete example of modern UI test automation using Java and Maven.

This framework includes automated UI tests for the website **practicesoftwaretesting.com**.
